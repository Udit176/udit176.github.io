Transparent RL: Explaining Agent Decisions

This project builds an interpretable reinforcement learning system using tabular Q-learning on FrozenLake. Instead of treating the policy as a black box, the system exposes how decisions are made by analyzing action values and confidence at every state.

Motivation

Reinforcement learning agents typically optimize behavior without providing insight into their decision process. This project addresses that gap by making the policy fully transparent, allowing us to understand not just what the agent does, but why.

Approach

  • Train a tabular Q-learning agent in a discrete gridworld (FrozenLake)
  • Extract the learned Q-values for each state-action pair
  • Rank actions per state and identify the optimal decision
  • Compute a confidence metric using the gap between top two actions

Key Outputs

Policy Map: shows the selected action in every state

Confidence Map: highlights how decisive each action is

Q-Value Heatmap: exposes the full decision structure

Takeaway

The project demonstrates that even simple RL systems can be made interpretable. By exposing action values and confidence, we can identify where the agent is certain, where it is ambiguous, and how its behavior evolves across the state space.
